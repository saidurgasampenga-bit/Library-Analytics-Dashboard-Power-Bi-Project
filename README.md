## 📖 Project Overview

The Library Analytics Dashboard is an interactive Business Intelligence solution developed using Microsoft Power BI. The project focuses on transforming raw library data into meaningful insights through data cleaning, data modeling, DAX calculations, and visual analytics.

The dashboard helps analyze book circulation, fine collection, branch performance, and borrowing trends, enabling efficient monitoring and data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze book usage across branches
- Track and monitor fine collection
- Identify branch-wise reading trends
- Measure library performance through KPIs
- Present insights using interactive visualizations

---

## 🛠 Data Cleaning & Transformation

The dataset contained several data quality issues, including:

- Missing Book IDs
- Inconsistent Branch Names
- Invalid Fine Amount Formats
- Null Values
- Data Standardization Requirements

These issues were resolved using Power Query to ensure accurate reporting and analysis.
